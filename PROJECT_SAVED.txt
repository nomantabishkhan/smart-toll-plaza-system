API ENDPOINTS (5 TOTAL)
===============================================================================

All require Django login. Available at:

  GET  /api/stats/live/
       Today's vehicle count by class

  GET  /api/stats/hourly/?date=2024-01-15
       24-hour hourly trend data

  POST /api/upload/video/
       Upload video for processing

  GET  /api/upload/status/?id=<uuid>
       Check video processing status (used for real-time polling)

  GET  /api/audit/export/?date=2024-01-15
       Generate daily audit report

COMMON QUESTIONS
===============================================================================

Q: How do I run the server?
A: python manage.py runserver 127.0.0.1:8000

Q: How do I access admin panel?
A: http://localhost:8000/admin/ (use superuser login)

Q: Why is inference stubbed?
A: Currently creates random detections for demo.
   Replace synthetic code with real YOLO parsing in tasks.py

Q: Can I use this for production?
A: Yes! Switch to PostgreSQL (USE_SQLITE=false in .env)
   See backend/README.md "Production Deployment" section

Q: Where do I edit the models?
A: backend/smarttoll/toll/models.py

Q: Where do I edit the API?
A: backend/smarttoll/toll/api.py

Q: How do I add more vehicle classes?
A: Use admin panel at /admin/ or see QUICK_REFERENCE.md for shell command

Q: Why is everything working without Docker?
A: Uses SQLite (local) + eager Celery (sync tasks)
   Both configured for Windows native execution
